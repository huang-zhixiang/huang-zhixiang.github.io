hexo后台常驻

发表于 2023-01-01 分类于 hexo博客 本文字数： 933

对于用xshell启动的服务 一旦xshell断开运行的服务也会同时断开，所以就需要一个可以后台运行的工具pm2

安装

npm install -g pm2

在linux下记得建立软连接

在hexo新项目中创建一个新文件名为hexo_run.js用于运行hexo博客

const { exec } = require('child_process')
exec('hexo server',(error, stdout, stderr) => {
        if(error){
                console.log('exec error: ${error}')
                return
        }
        console.log('stdout: ${stdout}');
        console.log('stderr: ${stderr}');
})

运行

cd到项目目录下

pm2 start hexo_run.js

pm2常用命令

查看进程状态

pm2 list

停止应用

pm2 stop id|name|all|json|stdin

其中，all 会将所有应用终止；id 为 pm2 list 查询出来的 id，name 为 pm2 list 查询出来的 name。其余两个暂时还不清楚

重启应用

pm2 restart id|name|all|json|stdin

参数含义同 pm2 stop

删除应用

pm2 delete name|id|script|all|json|stdin

参数含义同 pm2 stop

查看某个应用的信息

pm2 describe name|id

参数含义同 pm2 stop

pm2 start hexo_run.js   #启动
pm2 list    #查看pm2管理的所有服务

pm2 stop all    #停止pm2列表的所有服务
pm2 stop 0 #停止进程为0的进程

pm2 reload all #重新载入列表所有进程
pm2 reload 0 #重载列表中进程为0的进程

pm2 restart all #重启列表中所有的进程
pm2 restart 0   #重启列表中进程为0的进程

pm2 delete 0    #删除列表中进程为0的进程
pm2 delete all  #删除列表中所有的进程
# hexo部署
